# Sample Visa Packages

This directory contains sample PDF packages generated by the system for demonstration and testing purposes.

## Structure

### h1b/
Complete H-1B visa petition packages for various test cases.

**Latest versions:**
- `FINAL_H1B_PACKAGE_COMPLETE.pdf` - Final complete H-1B package
- `FERNANDA_SANTOS_H1B_COMPLETE_USCIS_PACKAGE.pdf` - Complete USCIS-ready package
- `PROFESSIONAL_H1B_PACKAGE_FERNANDA_SANTOS.pdf` - Professional formatted package

**Archive:** Old iterations are stored in `h1b/archive/`

### i539/
I-539 (B-2 Tourist Visa Extension) packages.

**Sample packages:**
- `Maria_Santos_I539_PACOTE_COMPLETO_USCIS.pdf` - Complete I-539 package
- `Roberto_Silva_Mendes_I539_COMPLETE_PACKAGE.pdf` - Another complete example
- `FINAL_COMPLETE_164_PAGES.pdf` - Comprehensive 164-page package

### i765/
I-765 (Employment Authorization Document) packages.

**Sample:**
- `Carlos_Eduardo_Ferreira_I765_EAD_COMPLETE_PACKAGE.pdf` - Complete EAD application

### archive/
Miscellaneous archived packages and test PDFs.

## Purpose

These sample packages serve multiple purposes:

1. **Quality Assurance** - Verify package generation works correctly
2. **Demonstrations** - Show potential clients what the system produces
3. **Reference** - Developers can see expected output format
4. **Testing** - Validate PDF structure and content

## Notes

- All sample packages use fictional applicant data
- Packages are generated by scripts in `scripts/generators/`
- New packages should follow existing naming conventions
- Archive old iterations to keep directories clean
